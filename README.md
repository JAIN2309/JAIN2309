## 🔐 Security Philosophy

<div align="center">

| What | How |
|---|---|
| Password storage | Argon2id — 64MB memory, 3 iterations, parallelism 4 |
| Sensitive fields | AES-256-GCM field-level encryption (email, phone, license) |
| Session tokens | JWT with Redis blacklist — instant logout invalidation |
| Brute-force | Per-account rate limiting + IP-spray detection |
| Logs | Sensitive field redaction on all request logs |

*Security isn't a feature — it's the foundation.*
